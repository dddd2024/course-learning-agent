# Course Learning Agent v1.0.0

首个稳定源代码版本。

## 主要能力

- 课程、资料与学习任务管理。
- TXT、Markdown、PDF、DOCX 与 PPTX 资料解析。
- 基于课程资料的问答、引用与证据不足处理。
- 知识点提纲、知识图谱、测验、错题与薄弱点分析。
- 单课程与多课程学习计划、任务状态流转和历史记录。
- Agent 运行审计、真实模型元数据记录与脱敏证据验证。
- 用户级 OpenAI-compatible 模型配置。
- Windows 本地启动与停止脚本。

## 发布验收

本版本在发布前完成以下门禁：

- 后端完整测试通过。
- 前端类型检查、单元测试与生产构建通过。
- 数据库迁移 dry-run 与 smoke test 通过。
- Playwright 端到端测试通过，失败和跳过均为 0。
- V7 功能闭环验收全部通过。
- 两轮真实 LLM 验收均完成 6/6 场景，mock、fallback、degraded 和元数据缺失计数均为 0。
- 仓库 Git 索引卫生检查通过：不存在被追踪但应被 `.gitignore` 排除的文件，不存在 Python 运行时二进制、缓存、虚拟环境、数据库、日志、测试报告、构建目录、压缩包或未知二进制文件。
- 发布 tag 仅由完成 CI 的 `main` 提交自动创建。

## 仓库内容说明

GitHub Release 提供该版本对应的源代码快照。以下内容不属于源代码版本，已从 Git 索引中排除：

- Python `__pycache__`、`.pyc`、`.pyo`、`.pyd` 等运行时产物；
- 虚拟环境、Node.js 依赖和前端构建结果；
- SQLite 数据库、用户上传资料和解析产物；
- 测试日志、Playwright 报告、临时 evidence 目录和本地缓存；
- `.env`、凭据及其他本机配置。

运行项目时，请根据 `backend/.env.example` 创建本地环境配置，并按仓库文档安装后端与前端依赖。
